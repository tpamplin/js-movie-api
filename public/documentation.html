MyFlix Documentation

Introduction

This api can be used to find information about different movies, directors, and genres.

You can:

Get all movies as a JSON object

Get data about a specific movie, director or genre as a JSON object

Add and remove users

Update user profiles

Add and remove movies from a user's list of favorites

Endpoints Overview

Request Type Request Body Response Endpoint Description
GET None. JSON /movies Returns a JSON object containing a list of all movies.
GET None. JSON /movies/:Title Returns a JSON object containing all information about a specific movie.
GET None. JSON /genres/:Name Returns a JSON object containing data about a specific genre.
GET None. JSON /directors/:Name Returns a JSON object containing information about a specific director.
POST JSON object with New User Data 201 "User Successfully Created" /users Adds a new user to the list of users.
PUT JSON object with updated data 201 "Username Changed Successfully" /users/:id/username Updates the username to the new username the user has picked.
POST JSON object with new movie data 201 "Movie added to favorites" /users/:id/favorites Adds a movie to a user's list of favorite movies.
DELETE None. 201 "Movie Removed from favorites" /users/:id/favorites Removes a movie from a user's favorite list.
DELETE None. 201 "User deleted" /users/:id Removes a user from the list of users.

PUT a new username

/users/:Username

A PUT request to this url will update a specific user's username with a new username given by the user.

Request Body Example Response Body Example
{"Username": "WallyBoy"} Status Code 201. Username updated successfully:
{
"_id": "66bbb79ef89578c9fe228fb5",
"Username": "WallyBoy",
"Password": "Password123!",
"Email": "wollonof@bigwollonof.com",
"Birthday": "1989-04-27T00:00:00.000Z",
"Favorites": [
"66bba02c90e880a8cf228fb9", "66bba48590e880a8cf228fbd", "66bb983190e880a8cf228fb6", "66bba1b390e880a8cf228fbb" ] }
Parameters:

This request takes the username parameter. Replace ':Username' in the url with the current username of the user whose username you are trying to change, the api should change the username and then send back the new updated user object. For example, a PUT request to:

/users/wollonof

along with a request body containing the new username, would return the json object above

Post A Movie To A User's Favorites

/users/:Username/favorites

A POST Request to this url will add a specific movie to a specific user's favorites list.

Examples

Request Body Example Response Body Example
{"MovieID": "66bba02c90e880a8cf228fb9"} Status Code 201, movie added to favorites successfully.
{
"_id": "66defb14cef558418a56ff61",
"Username": "BigTugg",
"Password": "Tuggerino1234",
"Email": "big@tugg.com",
"Birthday": "1976-04-23T18:25:43.511Z",
"Favorites": [
"66bba02c90e880a8cf228fb9"
],
"__v": 0
}
Parameters:

This request takes the username parameter. Replace ':Username' in the url with the id of the user whose favorites list you are trying to update, the api should add the movie to the user's favorites list, and then respond with the object it just added. For example, a POST request to:

/users/12/favorites

along with the appropriate request body, containing the new ObjectId of the movie that you wish to add to the list, would add that movie to the list and return the json object above

DELETE A Movie from A User's Favorites

/users/:id/favorites/:title

A DELETE request to this url will remove a movie from a specific user's favorites

Examples

Request Body Example Response Body Example
{"MovieID": "66bba02c90e880a8cf228fb9"} { "_id": "66defb14cef558418a56ff61", "Username": "BigTugg", "Password": "Tuggerino1234", "Email": "big@tugg.com", "Birthday": "1976-04-23T18:25:43.511Z", "Favorites": [], "__v": 0 }
Parameters:

This request takes the Username parameter. Replace ':Username' in the url with the username of the user whose favorites list you are trying to update, and add the id of the movie that you want to remove to the request body. for example a DELETE request to:

/users/BigTugg/favorites

along with the request body shown above, will respond with the json object shown above.

DELETE A User

/users/:Username

A DELETE request to this url will delete a user from the system

Examples

Request Body Example Response Body Example
None. Status Code 201: BigTugg was deleted
Parameters:

This request takes the id parameter. Replace ':Username' in the url with the id of the user who you are deleting. The api should delete the user, and then respond with a confirmation status code. For example, a DELETE request to:

/users/BigTugg

Will delete the user with the Username BigTugg and return:

Status Code 201: BigTugg was deleted.
