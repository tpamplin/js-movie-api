MyFlix Documentation

Introduction

This api can be used to find information about different movies, directors, actors, and genres. View in a list or search for a specific movie to see more information about it. You can create a profile, so that you can save your favorite movies, etc... to lists of favorites.

Endpoints

Request Type Request Body Response Endpoint Description
GET None. JSON /movies This will return a JSON object containing a list of all movies
GET None. JSON /movies/:id This will return a JSON object containing all information about a specific movie
GET None. JSON /genres/:name Returns a JSON object containing a description of the genre and a list of movies in that genre.
GET None. JSON /directors/:name Returns a JSON object containing information about a specific director.
POST JSON object with New User Data 201 "User Successfully Created" /users Adds a new user to the list of users
PUT JSON object with updated data 201 "Username Changed Successfully" /users/:id/Username Updates the username to the new username the user has picked
POST JSON object with new movie data 201 "Movie added to favorites" /users/:id/favorites add a movie to a user's list of favorite movies.
DELETE None. 201 "Movie Removed from favorites" /users/:id/favorites/:name Removes a movie from a user's favorite list.
DELETE None. 201 "User deleted" /users/:id Removes a user from the list of users
